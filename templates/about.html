About Us

Mission: Innovating secure identity verification with advanced face liveness detection.

Project Goal: Develop a browser-based face authentication platform for Aadhaar holders, accessible on desktops, mobiles, and tablets.

Challenge: Create ML models to detect face liveness and prevent spoofing (e.g., photo-of-photo, video attacks) using passive, active, or hybrid methods.

Technical Focus:
- Use computer vision for liveness detection within the browser.
- Ensure detection within 500ms for a seamless user experience.
- Maintain a model size under 5MB for efficient downloads and performance.

Outcome: Deliver a robust solution demonstrating effective face liveness detection in a browser environment.
